### Points of interest

Features of this website that I have put some intention behind:

  - Less and SCSS are now fading in favor of nested CSS rules, so I've used those here
  - Header
    - Shrinks both horizontally and vertically on viewport resize 
    - The focal point of the header is the little shelter, which is honored across all screen sizes
  - Dark mode. Thank goodness we have functions for this now. [Code blocks](/2025/11/30/part1.html#keep-your-function-short-and-simple) also look good.
  - Responsive layout of the side and top menus
